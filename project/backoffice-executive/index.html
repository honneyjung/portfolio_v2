경영진 관리자

퍼블리싱 인덱스 가이드 안내

  1. Depth 1 ~ 4 항목으로 이루어져 있으며, Depth 1 ~ 3은 페이지, Depth 4는 팝업입니다.
    • Depth는 서비스 플로우를 나타냅니다.
  2. html 항목 < td > 태그에는 < a > 태그만 넣어주세요.
  3. 날짜 항목 < td > 태그에는 하단 텍스트만 기입해 주세요. (작업 완료 후 프론트 전달 전 날짜 기입 부탁드려요!)
    • 날짜 : YYMMDD (사이 " - "는 필요 없습니다.)
    • 작업 필요 : TBD
    • 미사용 : x
  4. 상태 항목 < td > 태그는 날짜 항목 텍스트에 따라 자동 기입됩니다.
    • YYMMDD : 작업 완료
    • TBD : 추후 작업 필요
    • x : 미사용
  5. comment 항목 < td > 태그에는 구두 합의된 내용이나 히스토리가 필요한 내용을 "[YYMMDD] 내용" 형식으로 기입해 주세요.
    • ex : [240710] homezone이 부릅니다. 책을 넘기는 듯한 마음으로
    • ex : [240710] 윤딴딴이 부릅니다. 겨울을 걷는다
    • ex : [240911] 빈센트블루가 부릅니다. 비가와

유의사항

< colgroup > 태그는 첫 번째 < .tableWrap > 안 < table > 태그에만 존재해야 합니다.

공지사항

반응형 화면인 경우 < td >태그에 .onlyPcMo 클래스 추가 필요

PC 화면인 경우 < td >태그에 .onlyPc 클래스 추가 필요

모바일 화면인 경우 < td >태그에 .onlyMo 클래스 추가 필요

페이지수가 늘어남에 따라 네비게이션 메뉴는 대시보드에서만 관리하고, 이외 페이지에서는 영역만 잡아둔다.
